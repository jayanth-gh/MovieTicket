How to Access MovieHub

Access from the Same Network

To access MovieHub from other devices on your network (like phones or other laptops), follow these steps:

  1. Make sure your XAMPP server is running (Apache and MySQL)
  2. Find your computer's IP address:
    • On Windows: Open Command Prompt and type ipconfig
    • On Mac/Linux: Open Terminal and type ifconfig or ip addr
  3. Look for "IPv4 Address" - it will look like 192.168.1.xxx
  4. On other devices, open a web browser and enter: http://192.168.1.xxx/movie-booking (replace with your actual IP)

Note: All devices must be connected to the same Wi-Fi network.

Make Website Public (Advanced)

To make your website accessible from anywhere on the internet, you have several options:

  1. Port Forwarding: Configure your router to forward port 80 to your computer
  2. Use a Dynamic DNS service: Services like No-IP or Dynu can give you a public domain name
  3. Deploy to a hosting service: Upload your website to a hosting provider like:
    • GitHub Pages (free)
    • Netlify (free tier available)
    • Vercel (free tier available)
    • Traditional web hosting (paid)

Security Warning: Making your XAMPP server publicly accessible can pose security risks. Consider using a proper hosting service for public websites.
